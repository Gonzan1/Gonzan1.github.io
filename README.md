## 🛠️ About this project
This repository contains the source code for my personal website, built for Lab 01. It introduces who I am and demonstrates the use of:
* **HTML5 & CSS3** for structure and custom styling.
* **Bootstrap 5** for a fully responsive layout and grid system.
* **JavaScript** for interactivity, including a functional Dark Mode toggle and a dynamic user-input section.

## ✨ Features
* Responsive design (Desktop & Mobile).
* Custom CSS styling and font integration.
* Interactive "Dark Mode" theme toggle.
* Dynamic DOM manipulation using JavaScript to display user input.
